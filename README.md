# 💹 Stock Portfolio Dashboard

A web-based dashboard built with **Streamlit** to track stock prices and technical indicators in real-time.

## Features
* **Real-time Price Tracking:** Get the latest stock prices using `yfinance`.
* **Technical Analysis:** Calculate RSI, EMA (50/200), and MACD.
* **News Integration:** Automatically fetch the latest news for your selected tickers.

## How to Run
1. Clone the repository: `git clone https://github.com/ronnieshek/stock-dashboard.git`
2. Create and activate a virtual environment.
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `streamlit run app.py`

## Technologies Used
* Python 3.13
* Streamlit
* yfinance
* pandas_ta
